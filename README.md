# CodeAlpha C++ Projects

This repository contains my C++ internship projects completed as part of the **CodeAlpha Internship Program**.

## Projects Included

### 1. CGPA Calculator
A console-based application that calculates the CGPA of a student.

#### Features
- Enter number of courses.
- Input grade points and credit hours.
- Calculate total grade points.
- Calculate and display final CGPA.

---

### 2. Login and Registration System
A simple user authentication system using C++.

#### Features
- User Registration
- User Login
- Username and Password Validation
- Displays successful or failed login messages.

---

### 3. Banking System
A basic banking management system developed using Object-Oriented Programming concepts.

#### Features
- Create Account
- Deposit Money
- Withdraw Money
- Display Account Details
- Balance Management

---

## Technologies Used

- C++
- Object-Oriented Programming (OOP)
- Console Application
- Visual Studio 2015

---

## How to Run

1. Clone this repository.
2. Open the project in Visual Studio.
3. Build the project.
4. Run the application.
